*#Disk Dock Lock*

Disk Dock Lock is an encryption program for data storage devices (HDD, SSD, USB, CD, DVD). The program can protect your data storage devices using AES standards. You can also transfer data to your storage device before locking it, or load a disk image file (ISO, IMG) before locking. 



Net Byte Ware© accepts no responsibility. Before locking your data storage devices,
three "Are you sure?" messages will appear. You can reject one of these messages to cancel the process.
